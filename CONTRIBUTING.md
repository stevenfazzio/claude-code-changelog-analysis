# Contributing

Thanks for your interest in this project! Bug fix PRs are welcome, and feature ideas are appreciated as issues.

## Bug fixes

If you've found a bug, feel free to open a PR with a fix. Please include a clear description of the problem and how your change addresses it.

## Feature requests

Have an idea? Open an [issue](https://github.com/stevenfazzio/claude-code-changelog-analysis/issues) and describe what you'd like to see. I may not implement everything, but I'd like to hear your suggestions.

## Running the pipeline

The pipeline has stages that call external APIs. You'll need your own API keys to run the full pipeline locally:

- `ANTHROPIC_API_KEY` — for the enrich stage (Claude)
- `CO_API_KEY` — for the embed stage (Cohere)
- `gh` CLI authenticated — for the fetch stage

See the [README](README.md) for full setup instructions.

## Important notes

- Files in `docs/` are auto-generated by CI. Don't commit changes to them on feature branches — the pipeline will regenerate them after merge.
- Data files (`data/*.parquet`) are stored with Git LFS. Make sure you have [Git LFS](https://git-lfs.com/) installed.
